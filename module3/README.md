# 美股估值系统（个人学习版）

输入 Ticker，输出估值区间 + 关键假设 + 情景分析 + 告警。数据来源：Yahoo Finance，本地网页：Streamlit。

## 运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 估值体系概览

本系统集成三类估值方法，形成"内在价值为锚、相对估值验证"的分析框架。

| # | 模型 | 核心逻辑 | 适用场景 | 需要估计的参数 |
|---|------|----------|----------|----------------|
| 1 | **PE 回归估值** | 用 EPS 增速、Beta、派息率回归拟合"应得PE"，乘以稀释EPS | 盈利型公司（科技、金融、消费防御） | gEPS、行业无杠杆Beta |
| 2 | **内在价值折现** | 将未来自由现金流（或股利）折现至今，自动选 FCFF / FCFE / DDM | 通用（系统根据财务特征自动选择子模型） | Rf、ERP、违约利差、高速增长率、增长年数、永续增长率 |
| 3 | **相对估值** | 用 WACC、再投资率、利润率从基本面推导 EV 倍数，而非主观拍值 | 作为内在价值的置信区间验证，不作为单独结论 | 无需额外参数（由模型二 WACC 自动传入） |

**聚合规则**：估值中枢 = 内在价值折现结果；相对估值若与内在价值偏差 > 20%，系统输出告警，提示复核 WACC 或增长率假设。

## 模型一：PE 回归估值

### 原理

PE 倍数不是固定的"行业经验值"，而是由公司基本面决定的。通过对全市场上市公司进行横截面回归，可以得到"给定基本面的企业应当享有多少倍 PE"。

**2025年1月回归方程**（来源：Damodaran, NYU）：

```
PE = 24.17 - 1.07 × Beta + 53.16 × gEPS + 1.08 × PayoutRatio
目标价 = PE × 稀释EPS
```

- `Beta` 越高（风险越大）→ 应得 PE 越低
- `gEPS` 越高（成长越快）→ 应得 PE 越高（系数最大，影响最显著）
- `PayoutRatio` 越高（派息越多）→ 应得 PE 略高

### Beta 加杠杆公式

```
β_levered = β_unlevered × (1 + (1 - t) × D/E)
```

系统优先使用用户输入的行业无杠杆 Beta，自动加上公司的财务杠杆；若未输入，退而使用 yfinance 提供的回归 Beta。

### 适用场景

盈利稳定、EPS 可预测的公司。行业：科技、通信、医疗、消费防御、金融。

### 需要用户估计的参数

| 参数 | 如何获取 | 关键程度 |
|------|----------|----------|
| **gEPS（预期EPS增速）** | 卖方一致预期、历史 EPS CAGR、管理层指引 | ★★★ 最关键 |
| **行业无杠杆 Beta** | damodaran.com → Betas by Sector（每年1月更新） | ★★ 必须核对 |
| 税率 | 美国通常取 25%（联邦+州税） | ★ 一般用默认值 |

## 模型二：内在价值折现（FCFF / FCFE / DDM）

### 自动选择逻辑

系统根据公司财务特征自动选择最合适的子模型：

| 判定条件 | 选择子模型 | 原因 |
|----------|------------|------|
| 派息率 > 70% **且** 债务/市值 < 50% | **DDM** | 股息稳定可预测，直接折现股利 |
| 债务/市值 > 80% **或** 净利润 ≤ 0 | **FCFF** | 高杠杆或亏损时，股权层面现金流失真 |
| 其余情况 | **FCFE** | 杠杆稳定、盈利正常的一般企业 |

### 子模型 A：FCFF（公司自由现金流）

评估整盘生意的价值（企业价值 EV），不区分债股来源。

**FCFF 计算**：
```
FCFF = EBIT × (1 - t) - Net CapEx - ΔNon-cash WC
  Net CapEx  = CapEx - 折旧摊销
  ΔWC        = 当年营运资本 - 上年营运资本
```

**折现率：WACC**：
```
re   = Rf + β_levered × ERP             （股权成本，CAPM）
rd   = Rf + Default Spread              （税前债务成本）
WACC = re × E/(D+E) + rd × (1-t) × D/(D+E)
```

**企业价值 → 每股权益**：
```
EV       = Σ FCFF_t/(1+WACC)^t + TV/(1+WACC)^n
TV       = FCFF_n × (1+g_stable) / (WACC - g_stable)
每股价值 = (EV - 总债务 + 现金) / 稀释股本
```

**额外需要**：违约利差（用于计算 rd）。

### 子模型 B：FCFE（股权自由现金流）

扣除债权人后，真正属于股东的自由现金流。

**FCFE 计算**：
```
FCFE = 净利润 + D&A - CapEx - ΔWC + 净债务融资额
```

> ⚠️ 净债务融资额（借入 - 偿还）是关键变量：大量借入 → FCFE 飙升；大额还债 → FCFE 可能为负。

**折现率**：仅用 `re`（股权成本），不用 WACC。

```
权益价值 = Σ FCFE_t/(1+re)^t + TV/(1+re)^n
每股价值 = 权益价值 / 稀释股本
```

### 子模型 C：DDM（股利贴现）

公司只值它未来能给你的现金股利。

```
价值 = Σ DPS_t/(1+re)^t + TV/(1+re)^n
TV   = DPS_n × (1+g_stable) / (re - g_stable)
```

**硬约束**：`g_stable ≤ Rf`，系统自动强制。

### 需要用户估计的参数

| 参数 | 如何获取 | 用于哪些子模型 | 关键程度 |
|------|----------|----------------|----------|
| **无风险利率 Rf** | 10年期美债收益率，查询 FRED 或 Bloomberg | FCFF / FCFE / DDM | ★★★ |
| **股权风险溢价 ERP** | damodaran.com → Implied ERP（每月更新） | FCFF / FCFE / DDM | ★★★ |
| **高速增长率 g_high** | 卖方一致预期、历史收入/FCF CAGR | FCFF / FCFE / DDM | ★★★ |
| **高速增长年数 n** | 主观判断：成熟公司 5 年，成长型 7–10 年 | FCFF / FCFE / DDM | ★★ |
| **永续增长率 g_stable** | 约等于名义 GDP 增速，约 2–3%，且必须 ≤ Rf | FCFF / FCFE / DDM | ★★★ |
| **违约利差 Default Spread** | 查公司信用评级对应利差表 | **仅 FCFF** | ★★（FCFF专用） |
| 税率 | 美国通常 25% | FCFF / FCFE | ★ 一般用默认值 |

## 模型三：相对估值（基本面推导倍数）

### 核心哲学

> "倍数是 DCF 的浓缩版——任何合理的倍数都可以从基本面推导，主观拍一个 12× 是不严谨的。"

倍数不是主观经验值，而是由基本面决定：

### EV/EBITDA 推导

```
EV/EBIT = (1 - t) × (1 - 再投资率) / (WACC - g)
（注：此处以 EBIT ≈ EBITDA 作近似）

再投资率 (RR) = Net CapEx / NOPAT = (CapEx - D&A) / (EBIT × (1-t))
```

### EV/Sales 推导

```
EV/Sales = 税后EBIT利润率 × (1 - RR) × (1 + g) / (WACC - g)
```

这两个公式本质上是 Gordon 增长模型（永续增长 FCFF）的变形，所用 WACC 和 g 直接来自内在价值折现模型的计算结果。

### 用途

- 不作为独立估值结论，而是验证内在价值的合理性
- 若 EV/EBITDA 与内在价值偏差 > 20% → 系统告警，提示复核 WACC 或增长率
- 可反向推算：当前市值隐含的再投资率和利润率是否现实？

### 需要用户估计的参数

**无需额外输入**——WACC、g、税率来自内在价值折现模型；CapEx、D&A、EBIT 从财报自动读取。

## 目录结构

```
src/valmod/         核心八层流水线
  data_layer/       Layer 1: 数据拉取（fetch.py）
  normalization/    Layer 2: 标准化
  classification/   Layer 3: 模型选择（含行业推荐）
  assumptions/      Layer 4: 假设引擎
  models/
    multiples.py    基本面推导 EV/EBITDA 和 EV/Sales（模型三）
    damodaran_iv.py 内在价值折现，含 PE 回归（模型一、二）
  aggregation/      模型融合（IV锚定，相对估值验证）
  warnings/         告警系统
app.py              Streamlit 网页界面
```
